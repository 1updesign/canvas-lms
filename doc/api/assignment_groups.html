Assignment Groups API

API for accessing Assignment Group and Assignment information.

Actions

GET /api/v1/courses/:course_id/assignment_groups.json AssignmentGroupsController#index

Returns the list of assignment groups for the current context. The returned groups are sorted by their position field.

API method argument:

  • include[]
    "assignments"
    Associations to include with the group.

Example Response:

?include[]=assignments

[
  {
    "position": 7,
    "name": "group2",
    "id": 1,
    "assignments": [...]
  },
  {
    "position": 10,
    "name": "group1",
    "id": 2,
    "assignments": [...]
  },
  {
    "position": 12,
    "name": "group3",
    "id": 3,
    "assignments": [...]
  }
]

API response field:

  • id

    The unique identifier for the assignment group.

  • name

    The name of the assignment group.

  • position
    Integer
    The sorting order of this group in the

    groups for this context.
